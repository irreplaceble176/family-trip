## 🚀 方法一：Netlify（最簡單，拖曳上傳）

1. 開啟 https://netlify.com，點「Sign Up」→ 用 Google 或 GitHub 帳號登入
2. 登入後進入 Dashboard，找到畫面下方：
   **「Deploy manually」** 或 **「drag and drop」** 區域
3. 把整個 `pwa-package` 資料夾**直接拖進去**
4. 等 30 秒，Netlify 會給你一個 `https://xxx.netlify.app` 網址
5. 用手機 Chrome 開啟該網址，瀏覽器會自動跳出「安裝 App」提示

---

## 🚀 方法二：GitHub Pages

1. 到 https://github.com，登入或註冊帳號
2. 點右上角 **「+」→「New repository」**
3. Repository name 填 `trip-ledger`，選 **Public**，按「Create repository」
4. 進入 repo 頁面，點 **「uploading an existing file」**
5. 把 `pwa-package` 裡面**所有檔案**（含 icons 資料夾）拖上去上傳
6. 點「Commit changes」
7. 進入 **Settings → Pages → Branch → 選 main → Save**
8. 等 1-2 分鐘，網址會是：
   `https://你的帳號.github.io/trip-ledger/`
9. 用手機 Chrome 開啟即可安裝

---

## 📱 手機安裝步驟

### Android（Chrome）
1. 用 Chrome 開啟網址
2. 右上角選單（⋮）→「新增至主畫面」或「安裝應用程式」
3. 點「安裝」，桌面出現圖示即完成

### iPhone（Safari）
1. 用 **Safari** 開啟網址（Chrome 在 iOS 上無法安裝 PWA）
2. 底部分享按鈕（□↑）→「加入主畫面」
3. 名稱確認後點「加入」

---

## ⚠️ 注意事項
- 資料存在手機本機（localStorage + IndexedDB），換手機或清瀏覽器資料會消失
- 附件圖片存在 IndexedDB，同樣僅限本機
- 離線完全可用，不需要網路
